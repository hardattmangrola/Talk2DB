# Talk2DB: Intelligent Database Assistant

Talk2DB is a powerful, AI-driven application that allows users to interact with their MySQL databases and CSV files using natural language. It features a modern, professional interface with advanced analytics and reporting capabilities.

## Key Features

- **Natural Language to SQL**: Query your MySQL database using simple English (or other supported languages).
- **CSV Analysis**: Upload and analyze CSV files. The AI can determine relationships between multiple files and answer complex questions.
- **Advanced Admin Dashboard**: 
  - **Schema Management**: Build and manage your database schema visually.
  - **Advanced Analytics**: Detailed statistical summaries including 5-number summaries and outlier detection.
  - **AI Strategic Insights**: Get high-level analysis and patterns generated by Gemini AI.
  - **Interactive Visualizations**: View distributions, trends, and correlations through dynamic charts.
- **Rich Reporting**: Export comprehensive PDF or Excel reports containing statistical data, AI insights, and visualizations.
- **Monochrome Glassmorphism UI**: A sleek, high-end "Quantum" design with a monochromatic black, gray, and white color scheme.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide React, Recharts.
- **Backend**: Flask, Python, Pandas, ReportLab, Matplotlib, Seaborn.
- **AI**: Google Gemini API.
- **Database**: MySQL.

## Getting Started

### Backend Setup
1. Navigate to `backend/`
2. Create a `.env` file with:
   ```env
   GEMINI_API_KEY=your_key
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=your_password
   DB_NAME=your_db
   UPLOAD_FOLDER=./uploads
   ```
3. Install dependencies: `pip install -r requirements.txt`
4. Run the server: `python app.py`

### Frontend Setup
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

## Usage

- **Login**: Use `admin/admin123` for full access to the Dashboard and Analytics.
- **Chat**: Upload CSVs or query the database in the main chat interface.
- **Analytics**: Visit the Admin Dashboard and select the "Analytics & Reports" tab to perform deep data dives.
